# 行動規範 (Code of Conduct)

## 私たちの誓い

Penguin Control Centerプロジェクトは、年齢・体格・障害の有無・民族・性自認と性表現・経験レベル・
国籍・容姿・人種・宗教・性的指向などに関わらず、すべての人にとってハラスメントのない参加体験を
提供することを誓います。

## 私たちの基準

好ましい行動の例:

- 思いやりと優しさを持って接すること
- 異なる意見・視点・経験を尊重すること
- 建設的なフィードバックを行い、また快く受け入れること
- 誤りを認め、影響を受けた人々に謝罪し、その経験から学ぶこと

許容されない行動の例:

- 性的な言葉や画像の使用、性的な誘いかけ
- 荒らし行為、侮辱的・軽蔑的なコメント、個人・政治的な攻撃
- 公的・私的を問わないハラスメント
- 他者の住所やメールアドレス等、本人の同意なしに個人情報を公開すること

## 適用範囲

本行動規範は、プロジェクトの全スペース（Issue、Pull Request、Discussions等）、および個人が
プロジェクトを公式に代表する場面に適用されます。

## 運用

不適切な言動を目撃・経験した場合は、リポジトリのメンテナ（[@RT2231](https://github.com/RT2231)）まで
GitHub Issueまたは[Security Advisory機能](../../security/advisories/new)経由でご連絡ください。
メンテナはすべての苦情を迅速かつ公正に確認・調査します。
