# Pi-hole

## 概要

Pi-holeはDNSレベルで広告・トラッカーをネットワーク全体からブロックするソフトウェアです。
本プラグインはブロック状態の確認・一時停止/再開・ブロックリスト更新をGUIから行えるようにします。

## 主なCLIコマンド

| コマンド | 説明 |
|---|---|
| `pihole status` | 現在ブロック中かどうかを表示 |
| `pihole disable 300` | 300秒（5分）だけ一時的にブロックを停止 |
| `pihole enable` | ブロックを再開 |
| `pihole -g` | ブロックリスト（gravity）を最新版に更新 |
| `pihole restartdns` | DNS(FTL)サービスを再起動 |

## 設定ファイル

- パス: `/etc/pihole/setupVars.conf`
- 主要項目: `PIHOLE_INTERFACE`（監視インターフェース）、`IPV4_ADDRESS`
- **注意**: このファイルは他の設定ファイルと異なり`pihole -r`（再設定ウィザード）経由での変更が
  推奨されています。直接編集は上級者向けです。

## よくあるトラブル

- **`pihole`コマンドが権限エラー**: 多くのサブコマンドは内部で root 権限を必要とします。
  本プラグインでは変更系アクションに`privileged: true`を付与しpkexec経由で実行します
- **ブロックリスト更新が失敗する**: インターネット接続、または上流のブロックリストURLの到達性を確認
- **一部の広告がブロックされない**: CDN配信の広告やHTTPS化された追跡スクリプトはDNSブロックでは
  防げない場合があります（ブラウザ拡張との併用を推奨）
